## 关于 Shannon

Shannon 是一个企业级 AI 代理编排器，提供：

- 🤖 **多模型支持**：集成 OpenAI、Anthropic、Google、DeepSeek 等
- 🔄 **智能路由**：根据任务复杂度自动选择模型
- 🛡️ **安全执行**：基于 WASI 的 Python 代码沙箱
- 📊 **企业级特性**：完整的监控、成本控制和审计
